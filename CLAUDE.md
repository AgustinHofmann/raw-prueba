# RAW Design

App web de diseño de indumentaria especializada para diseñadores de moda.

## Stack
- React 19 + TypeScript
- Vite
- Fabric.js (canvas/dibujo)

## Etapa 1 — Scope
- Mockups de prendas 2D flat (remera, hoodie, pantalon)
- Aplicar colores/estampados a zonas del mockup
- Dibujar/trazar libremente sobre el mockup
- Exportar como PNG o JPG

## Estructura
```
src/           codigo fuente React
public/mockups/  SVGs de las prendas (tshirt.svg, hoodie.svg, pants.svg)
```

## Comandos
```bash
npm install    instalar dependencias
npm run dev    servidor de desarrollo
npm run build  build de produccion
```
